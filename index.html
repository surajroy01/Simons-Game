Simon Game

Press any key to start to start the game

Rules for the Simons Says Game

  • When game starts see which button is shows flash.
  • Click on the button which was flashed.
  • If you clicked correct button you will pass the Level and will move to the next level.
  • If not then start again.
  • When you reach Level 2 you have to click the previous button as well as current flashed button in sequence.
  • If you break the sequence you will lose and have to start again.
